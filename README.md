## 3. Issues & Solutions

### Issue 1: Could not find Extensions in Cursor
- Problem: I didn’t know where to install extensions  
- Solution: I searched using ChatGPT and learned the shortcut **Ctrl + Shift + X** to open Extensions

---

### Issue 2: Could not find where to log in Claude Code
- Problem: After installing Claude Code, I couldn’t see where to log in  
- Thought: I assumed I installed the wrong extension  
- Action: I uninstalled it and checked again with ChatGPT  
- Solution: I learned to use **Ctrl + Shift + P** (Command Palette) and opened Claude in the sidebar to log in  

---

### Issue 3: Installing Codex
- Problem: Similar confusion as Claude extension  
- Solution: Reused the same approach (search + command palette) to install and access it successfully
